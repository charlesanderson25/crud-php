# crud-php
Crud para cadastramento de cisternas

O sistema foi construído através do XAMP com o server Apache e o banco de dados MySQL. 

Para rodar esse projeto é necessário instalar o XAMP na sua máquina e inicializar o servidor Apache e o banco de dados MySQL. 

Após instalar o XAMP e inicializar os sistemas, será possível adicionar ou excluir cisternas pela página no front-end da aplicação acessando-a via: http://localhost/
